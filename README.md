# cpp-cli-hello-world

## Description
A POC for c++ on linux.

## Tech stack
- bash
- c++
- gcc 4
- g++ 9

## Docker stack
- docker-cli
- ubuntu:latest

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
